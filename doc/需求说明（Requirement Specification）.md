# 🧾 StudyCompanion – 需求说明（Requirement Specification）

## 1. 项目背景

大学生在学习过程中经常面对内容复杂、形式多样的课件（PPT、PDF等），但缺乏系统化整理与复习的工具。
 现有的大语言模型可以回答问题，却难以输出结构化、可复用、可控制风格的学习笔记。

**StudyCompanion**旨在成为一个智能化学习助手，帮助学生自动将课件转化为多种形式的高质量学习材料，覆盖“理解 → 整理 → 复习 → 测试”的完整学习闭环。

------

## 2. 整体目标

系统应能从课程讲义中自动提取内容，并依据用户需求生成不同风格、不同深度的学习笔记。
 用户可以自由控制笔记的详略与难易程度，也可以选择多种模板以适配不同学习阶段。

------

## 3. 核心功能板块

### 3.1 结构化笔记生成系统（Structured Note Generator）

系统的核心功能，用于将课件内容生成可调节的笔记。

- 提供**两条滑档控制**：
  - **详略程度**：简略 / 中等 / 详细
  - **难易程度**：简单 / 讲解型 / 学术型
- 用户可通过滑档组合生成9种风格笔记，从轻量复习到深度讲解。
- 输出结果保持统一格式：层级清晰、语义连贯、适合阅读或导出。

### 3.2 知识卡片（Knowledge Cards）

为期末复习与快速回顾设计。

- 卡片包含核心概念及解释、核心公式、核心算法等
- 支持按章节自动生成整套卡片。
- 输出结构固定、信息简洁、便于打印或导入复习工具。

### 3.3 模拟试题（Mock Exam Generator）

从课件与笔记内容中自动生成复习题目。

- 题型包括选择题、填空题、简答题三种，用户可以选择生成的题型。
- 每道题附参考答案与关键点解释。
- 支持生成章节测验或整卷模拟。

### 3.4 思维导图与知识树（Mind Map & Knowledge Tree）

以图形化方式展示章节、概念与知识之间的逻辑关系。

- 自动根据笔记层级生成节点与连接关系。
- 支持“展开/折叠”操作查看不同深度。
- 可导出为图像或交互式视图，用于复习导航。

### 3.5 页面式内容还原（Layout Reconstruction）

保留课件的视觉结构，提升学习沉浸感。

- 保留文字、图片、公式等原有布局。
- 插入截图和公式展示，自动补充说明文字。
- 每页生成结果均保持与原PPT的视觉分区一致。

### 3.6 浮动式问答助手（Floating Q&A Assistant）

在主界面右侧以悬浮球形式存在，供用户即时提问。

- 点击后弹出对话窗口，可就当前笔记内容进行提问。
- 不影响主界面操作，辅助学习过程。

------

## 4. 用户交互需求

1. **上传与解析流程**
   - 用户上传PPT文件后系统自动识别章节与内容。
   - 系统展示解析进度与预览缩略图。
2. **参数选择**
   - 用户通过滑档选择笔记风格（详略、难易）。
   - 可同时选择生成模板（笔记 / 知识卡 / 模拟题 / 思维导图）。
3. **生成与预览**
   - 系统展示生成进度条。
   - 生成完成后，用户可在同一页面浏览、编辑、切换风格。
4. **导出与保存**
   - 支持导出为Markdown、PDF或图片格式。
   - 用户可一键下载或保存到本地学习库。

------

## 5. 非功能性需求（简要）

| 类型         | 需求说明                                            |
| ------------ | --------------------------------------------------- |
| **响应速度** | 上传至笔记初稿生成时间不超过2分钟。                 |
| **兼容性**   | 优先支持PPTX格式，其次PDF；保证多页课件的完整解析。 |
| **稳定性**   | 若生成失败，可重新生成，不影响已生成内容。          |
| **可用性**   | 操作流程不超过三步（上传 → 设定 → 生成）。          |
| **一致性**   | 所有输出模板保持统一的结构层级与排版规范。          |

------

## 6. 需求优先级汇总

| 优先级 | 板块               | 说明                               |
| ------ | ------------------ | ---------------------------------- |
| 🔥 高   | 结构化笔记生成系统 | 系统核心功能，需实现滑档组合控制。 |
| 🔥 高   | 页面式内容还原     | 必须保持原PPT信息与图片布局。      |
| ⚙️ 中   | 知识卡片           | 期末复习主功能，需与笔记内容关联。 |
| ⚙️ 中   | 模拟试题           | 用于评估学习效果，内容来源于笔记。 |
| 🌿 低   | 思维导图与知识树   | 可作为增强展示模块。               |
| 🌿 低   | 浮动问答助手       | 保留为辅助功能，可后期接入。       |
